
Bienvenue dans la simulation de l'entrepôt d'Hamza et de Liora !
Notre projet peut être testé de 2 manières : par un fichier text ou par la console.

Simulation par fichier test.txt :

- ajouter au dossier taieb_touzani le fichier test
- lancer Eclipse (se placer dans le répertoire contenant le dossier taieb_touzani)
- importer notre projet (Import projects -> Existing projects into workspace -> indiquer dans "select root directory" le chemin menant au dossier taieb_touzani)
- compiler le programme (clic droit sur Main.java -> Run As -> Run Configurations -> Arguments -> ajoutez "test.txt" dans "Program arguments" -> run -> proceed)
- suivre les instructions

Simulation par console : 

- lancer Eclipse (se placer dans le répertoire contenant le dossier taieb_touzani)
- importer notre projet (Import projects -> Existing projects into workspace -> indiquer dans "select root directory" le chemin menant au dossier taieb_touzani)
- compiler le programme (clic droit sur Main.java -> Run As -> 1 Java Applications -> proceed)
- suivre les instructions

NB : Si vous testez la simulation par console après l'avoir testée par fichier, n'oubliez pas de supprimer le nom du fichier dans les arguments de compilation (clic droit sur Main.java -> Run As -> Run Configurations -> Arguments -> supprimer "test.txt" dans "Program arguments")

A bientôt !
